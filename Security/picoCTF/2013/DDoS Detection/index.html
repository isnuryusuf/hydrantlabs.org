picoCTF 2013: DDoS Detection (85)

Challenge:

It appears a SYN-flood style DDoS has been carried out on this system. Send us a list of the IP addresses of the attackers (in any order, separated by spaces), so we can track them down and stop them.

Pcap available to download here, or available to analyse online at CloudShark

Solution:

Looking at the info in the problem we need to filter by SYN packets which can be done via the following filter:

tcp.flags.syn==1

After running this filter we we get a list of IPs. Before compiling a list of attacker IPs we must eliminate legitimate users, those with a ACK response and our own IP:

199.59.148.147
74.125.228.64
128.237.255.81

After elminating packets with those source IPs we can make a nice space seperated list for submission:
