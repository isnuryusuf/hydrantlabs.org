picoCTF 2013: Pilot Logic (75)

Challenge:

You've gotten a partial dump of the disk from the hangar's machine, and you're pretty sure the pilot's password is cleverly hidden somewhere within it...

Solution:

The first step was to determine what the image was:

file pilot_image

Which resulted in

pilot_image: Linux rev 1.0 ext2 filesystem data

I then mounted the file and created a tree directory listing (shortened where needed) with hidden files visible:
